# Awesome Indonesian Dev Tools

A curated list of tools, libraries, and resources for Indonesian developers.

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

---

## Identity & Validation

### NIK (Nomor Induk Kependudukan)

- [nik-parser.rb](https://github.com/gizipp/nik-parser.rb) - Ruby gem: parse NIK, validate NPWP, phone, format Rupiah
- [nik-validator](https://github.com/riipandi/nik-validator) - NIK validator for JavaScript
- [validasi-nik](https://github.com/nicnocquee/validasi-nik) - Simple NIK validator

### NPWP (Nomor Pokok Wajib Pajak)

- [nik-parser.rb](https://github.com/gizipp/nik-parser.rb) - Includes NPWP validation and parsing

### Phone Numbers

- [nik-parser.rb](https://github.com/gizipp/nik-parser.rb) - Indonesian phone number validation (08xx, +62)

## Currency & Finance

- [rupiah](https://github.com/gizipp/rupiah) - Indonesian Rupiah toolkit. Format, parse, terbilang. AI-agent ready with MCP (JavaScript/TypeScript)
- [angka-rupiah-js](https://github.com/develoka/angka-rupiah-js) - Format angka ke Rupiah (JavaScript)
- [terbilang](https://github.com/riskihajar/terbilang) - Convert number to Indonesian words (PHP)

## Payment Gateway

- [veritrans-ruby](https://github.com/veritrans/veritrans-ruby) - Midtrans payment gateway client (Ruby)
- [veritrans-php](https://github.com/veritrans/veritrans-php) - Midtrans payment gateway client (PHP)
- [midtrans-node](https://github.com/veritrans/veritrans-node) - Midtrans payment gateway client (Node.js)
- [midtrans-go](https://github.com/veritrans/midtrans-go) - Midtrans payment gateway client (Go)
- [xendit-node](https://github.com/xendit/xendit-node) - Xendit payment gateway client (Node.js)
- [xendit-python](https://github.com/xendit/xendit-python) - Xendit payment gateway client (Python)
- [xendit-php](https://github.com/xendit/xendit-php) - Xendit payment gateway client (PHP)

## Regional Data & Geography

- [kodepos](https://github.com/emsifa/kodepos) - Indonesian postal code lookup API
- [wilayah-indonesia](https://github.com/kodewilayah/permendagri-72-2019) - Administrative area codes (Permendagri 72/2019)
- [idn-area](https://github.com/fityannugroho/idn-area) - Indonesian area data (provinces, regencies, districts, villages)
- [kode-wilayah-indonesia](https://github.com/emsifa/kode-wilayah-indonesia) - Indonesian area codes with API

## E-Commerce & Marketplace

- [tokopedia-api](https://github.com/nicnocquee/tokopedia-api) - Unofficial Tokopedia API client
- [tokopedia-scraper](https://github.com/nicnocquee/tokopedia-scraper) - Tokopedia product scraper

## Date & Time

- [hari-libur-indonesia](https://github.com/nicnocquee/hari-libur-indonesia) - Indonesian public holidays data

## ISBN & Publishing

- [isbn-toolkit-ruby](https://github.com/gizipp/isbn-toolkit-ruby) - ISBN validation, conversion, metadata lookup (Ruby)
- [isbn_toolkit](https://github.com/gizipp/isbn_toolkit) - ISBN toolkit for JavaScript/TypeScript
- [isbn-toolkit-php](https://github.com/gizipp/isbn-toolkit-php) - ISBN toolkit for PHP/Laravel

## NLP & Text Processing

- [sastrawi](https://github.com/sastrawi/sastrawi) - Indonesian stemmer (PHP)
- [SastrawiJS](https://github.com/nicnocquee/SastrawiJS) - Indonesian stemmer (JavaScript)
- [nlp-id](https://github.com/nicnocquee/nlp-id) - Indonesian NLP tools

## Government & Open Data

- [data.go.id](https://data.go.id/) - Indonesian government open data portal
- [api-indonesia](https://github.com/nicnocquee/api-indonesia) - Indonesian government API wrapper
