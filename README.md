# Awesome Indonesian Dev Tools

A curated list of tools, libraries, and resources for Indonesian developers.

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

---

## Identity & Validation

### NIK (Nomor Induk Kependudukan)

- [nik-parser.rb](https://github.com/gizipp/nik-parser.rb) - Ruby gem: parse NIK, validate NPWP, phone, format Rupiah

### NPWP (Nomor Pokok Wajib Pajak)

- [nik-parser.rb](https://github.com/gizipp/nik-parser.rb) - Includes NPWP validation and parsing

### Phone Numbers

- [nik-parser.rb](https://github.com/gizipp/nik-parser.rb) - Indonesian phone number validation (08xx, +62)

## ISBN & Publishing

- [isbn-toolkit-ruby](https://github.com/gizipp/isbn-toolkit-ruby) - ISBN validation, conversion, metadata lookup (Ruby)
- [isbn_toolkit](https://github.com/gizipp/isbn_toolkit) - ISBN toolkit for JavaScript/TypeScript
- [isbn-toolkit-php](https://github.com/gizipp/isbn-toolkit-php) - ISBN toolkit for PHP/Laravel

## Payment

- [veritrans-ruby](https://github.com/veritrans/veritrans-ruby) - Veritrans/Midtrans payment gateway client (Ruby)
- [veritrans-php](https://github.com/veritrans/veritrans-php) - Veritrans/Midtrans payment gateway client (PHP)
- [midtrans-node](https://github.com/veritrans/veritrans-node) - Midtrans payment gateway client (Node.js)

## Regional Data

- [kodepos](https://github.com/emsifa/kodepos) - Indonesian postal code lookup
- [wilayah-indonesia](https://github.com/kodewilayah/permendagri-72-2019) - Indonesian administrative area codes
